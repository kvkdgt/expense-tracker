# PWA Installation Guide

Spendly is now a Progressive Web App (PWA)! You can install it on your device and use it like a native app.

## How to Install

### On Mobile (Android/iPhone)

1. **Open Spendly in your mobile browser** (Chrome, Safari, etc.)
2. **Look for the install prompt** - You'll see an "Install App" button or banner
3. **Tap "Install"** or "Add to Home Screen"
4. The app will be installed and appear on your home screen
5. **Open it like a native app!**

### On Desktop (Chrome/Edge)

1. **Open Spendly in Chrome or Edge browser**
2. **Look for the install icon** in the address bar (usually a "+" or download icon)
3. **Click "Install"** when prompted
4. The app will open in its own window without browser UI

### Manual Installation (if prompt doesn't appear)

#### Android (Chrome):
1. Open the menu (three dots)
2. Select "Add to Home screen" or "Install app"
3. Confirm installation

#### iPhone (Safari):
1. Tap the Share button
2. Scroll down and tap "Add to Home Screen"
3. Customize the name if needed
4. Tap "Add"

## Troubleshooting

If the install prompt doesn't appear:
- Make sure you're using HTTPS (required for PWA)
- Clear browser cache and try again
- Check if your browser supports PWA installation
- Try the manual installation method above
